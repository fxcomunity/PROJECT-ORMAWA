readme_content = """# Ormawa Center - Sistem Manajemen Informasi Organisasi Mahasiswa

Aplikasi berbasis web **PHP Native** dan **Bootstrap** untuk manajemen data internal serta cetak kartu anggota Organisasi Mahasiswa (ORMAWA). Project ini disesuaikan dengan materi modul praktikum **Web Programming I** halaman 70 s/d 91.

## 🚀 Fitur Utama
1. **Autentikasi User (Session)**: Login & Registrasi Anggota (NIM digunakan sebagai username login).
2. **Form Registrasi 2 Kolom**: Tampilan form dinamis dan rapi yang terbagi menjadi data utama dan data pelengkap.
3. **Dashboard Konten Dinamis**: Menggunakan konsep shell template dengan routing halaman (`?page=nama_halaman`).
4. **Validasi Form**: Menggunakan JavaScript untuk validasi input sebelum dikirim ke server.
5. **Cetak PDF (FPDF)**: Fitur ekspor data menjadi bentuk fisik Kartu Anggota seukuran ID Card (KTP).
